# Interactive Portfolio Website

## Project Overview

This project is an interactive and responsive personal portfolio website developed using **HTML5, CSS3, and JavaScript**. The website showcases my personal profile, technical skills, blockchain projects, and contact information in a professional and user-friendly interface. The main objective of this project is to demonstrate the practical implementation of JavaScript concepts such as DOM manipulation, event handling, reusable functions, and form validation while creating an engaging user experience.

---

# Features

* Responsive and modern user interface
* Professional navigation bar with smooth scrolling
* Dynamic greeting message based on the current time
* Dark/Light mode toggle
* Show/Hide About section
* Interactive Project Gallery with image slider
* Previous and Next image navigation
* Automatic image slideshow
* Click image to view in full screen
* Interactive To-Do List
* Add and delete tasks
* Contact form with JavaScript validation
* Active navigation highlighting while scrolling
* Scroll progress indicator
* Back-to-top button
* Visitor counter
* Hover effects and animations
* Mobile-friendly responsive design

# DOM Manipulation Used

The project uses JavaScript DOM methods to interact with HTML elements.

Examples include:

* `document.getElementById()`
* `document.querySelectorAll()`
* `document.createElement()`
* `appendChild()`
* `classList.toggle()`
* `addEventListener()`
* `innerHTML`
* `style.display`

These methods are used to update webpage content dynamically and provide an interactive user experience.

# Responsive Design

The website is fully responsive and adapts to different screen sizes using CSS Media Queries. The layout automatically adjusts for desktop, tablet, and mobile devices while maintaining usability and visual consistency.

---

# How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (if not already installed).
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
6. The portfolio website will open in your default web browser.
